# Bitasmbl-Lightweight-Chat-App-No-Auth-6ae06f

## Description
Build a web application that allows users to join anonymous chatrooms and exchange messages in real-time using WebSockets, focusing on fast communication, a simple interface, and responsive updates without requiring user registration.

## Tech Stack
- Node.js
- Vue.js
- Socket.IO

## Requirements
- Build a web application that allows users to join anonymous chatrooms and exchange messages in real-time using WebSockets.
- Focus on fast communication.
- Simple interface.
- Responsive updates.
- No user registration.

## Installation
bash
git clone https://github.com/MrBitasmblTester2/Bitasmbl-Lightweight-Chat-App-No-Auth-6ae06f.git
cd Bitasmbl-Lightweight-Chat-App-No-Auth-6ae06f
npm install


## Usage
bash
npm start


## Implementation Steps
1. Initialize Node.js project and configure Socket.IO server.
2. Create Vue.js frontend for anonymous chatrooms and messaging.
3. Connect Vue.js client to Socket.IO server for real-time messages.
4. Implement room join and broadcast logic using WebSockets.
5. Ensure responsive UI updates without authentication.

## API Endpoints
- WebSocket connections managed via Socket.IO.
